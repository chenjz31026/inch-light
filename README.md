# inch-light
一个简单的记事本和多任务计时器



## 待新增功能

- [ ] 可新增桌面组件，专注一个，清单一个
- [ ] 专注/任务的名称为空的情况下点击新建任务时，会给出报错提示「请确定名称」，弹窗短暂出现后自然消失

## 更新日志



### v1.1.0

#### 功能新增：

- [x] 上导航栏美化
- [x] 清单模块中截止时间的颜色默认绿色，超时后才变成红色
- [x] 增加前台服务机制，切后台依然可继续倒计时

#### 模块新增：

【周期】模块：每天几时几分/每周第几天/每月第几天/每年几月几日定期提醒某事件，比如每月提醒机场的月卡该更新了，比如每周五充电宝该充电了、

- 按照时间先后罗列在屏幕上，不需要按周期频率划分子页面，统一呈现即可，但要标注出类型
- 右下角的新增按钮和其他模块一样，是黑色圆圈中有白色加号
- 每条提醒也可重命名、重新设定周期、左滑删除
